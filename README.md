## üìñ Example Files

### 01_basics.py
Covers fundamental Python concepts:
- Variables and data types (int, float, string, bool, None)
- Type conversion
- Basic arithmetic and string operations
- String formatting (f-strings, .format(), %)
- Variable naming rules

### 02_control_flow.py
Learn program flow control:
- If/elif/else statements
- For loops (with range, enumerate, nested loops)
- While loops
- Break and continue statements
- Practical examples

### 03_functions.py
Master function creation:
- Basic functions with parameters
- Default parameters
- Keyword arguments
- Variable arguments (*args, **kwargs)
- Lambda functions
- Recursion
- Variable scope
- Docstrings

### 04_data_structures.py
Explore Python's data structures:
- Lists (creation, indexing, slicing, methods, list comprehension)
- Tuples (immutable sequences)
- Dictionaries (key-value pairs, methods, nested dictionaries)
- Sets (unique elements, set operations)

### 05_file_handling.py
Work with files:
- Reading and writing files
- Different file modes
- CSV file operations
- JSON file operations
- File operations (exists, size, rename)
- Error handling with files

### 06_classes_objects.py
Object-oriented programming:
- Creating classes and objects
- Class vs instance attributes
- Methods (instance, class, static)
- Inheritance
- Encapsulation (private attributes)
- Special methods (__str__, __repr__, etc.)
- Properties (getters and setters)

### 07_error_handling.py
Handle errors gracefully:
- Try/except blocks
- Multiple exception handling
- Try/except/else/finally
- Raising exceptions
- Custom exceptions
- Assert statements
- Practical error handling examples

### 08_practical_projects.py
Complete projects combining multiple concepts:
- To-Do List Manager
- Number Guessing Game
- Contact Book
- Simple Calculator
- Password Generator
- Text Analyzer

### 09_modules_packages.py
Code organization:
- Importing built-in modules
- Common modules (math, datetime, random, json, etc.)
- Creating your own modules
- Module attributes
- Working with various standard library modules

## üí° Learning Path

For beginners, we recommend following this order:

1. Start with **01_basics.py** to understand fundamental concepts
2. Move to **02_control_flow.py** to learn how to control program flow
3. Study **03_functions.py** to create reusable code
4. Explore **04_data_structures.py** to work with collections
5. Learn **05_file_handling.py** to interact with files
6. Advance to **06_classes_objects.py** for OOP concepts
7. Master **07_error_handling.py** for robust programs
8. Practice with **08_practical_projects.py** to apply knowledge
9. Organize code with **09_modules_packages.py**
